
**Comprehensive guide to all Harvard CS50 courses**, including details, links, certificates, and **step-by-step instructions** to earn free certificates.

### Overview

Harvard's CS50 program offers high-quality computer science courses. Most provide **free certificates** via Harvard OpenCourseWare (OCW). Courses are updated annually (new version on January 1st).

**Key Points:**
- **Free Certificate**: Harvard OCW (no ID verification)
- **Paid Certificate**: edX (ID verified, $199–$449)
- Self-paced, no deadlines
- Requires free GitHub + edX accounts for submissions

### How to Get a Free Certificate (Step-by-Step)

1. **Create Accounts**
   - GitHub: [github.com/signup](https://github.com/signup)
   - edX (free audit): [edx.org/register](https://www.edx.org/register)

2. **Start the Course**
   - Go to the **OCW link** for your chosen course.
   - Begin with **Week 0** in the sidebar.

3. **Complete Weekly Work**
   - Watch lecture + read notes/transcript.
   - Do the problem set / project.
   - Submit via instructions on the page (usually GitHub + edX form).

4. **Score Requirement**
   - 70%+ on **all** problem sets and final project.
   - You can resubmit as many times as needed.

5. **Claim Certificate**
   - After finishing everything, go to [certificates.cs50.io](https://certificates.cs50.io/) or the course certificate page.
   - Enter your name → Download PDF (with verification link).

**Tips**:
- Use the **CS50 Duck AI** (`cs50.ai`) for help.
- Progress usually carries over when courses refresh in January.
- Courses are self-paced.
